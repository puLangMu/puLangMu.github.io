# Zimu Li Academic Homepage

Personal academic homepage for `https://pulangmu.github.io/`, built with the
`al-folio` Jekyll theme.

## Local Preview

Install dependencies:

```powershell
bundle install
npm ci
```

Run the site locally:

```powershell
bundle exec jekyll serve
```

Then open `http://127.0.0.1:4000/`.

## Deployment

The site is configured for GitHub Pages deployment through GitHub Actions.
After merging to `main`, set GitHub Pages to deploy from Actions if it is not
already configured that way.
